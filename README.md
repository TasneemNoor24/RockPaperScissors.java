# ✊✋✌ Rock Paper Scissors Game (Java)

A simple console-based Rock Paper Scissors game built using Java.  
This project is great for beginners to understand **if-else conditions, loops, and user input handling** in Java.

---

## 🎮 How the Game Works

- The user selects one option:
  - Rock ✊
  - Paper ✋
  - Scissors ✌

- The computer randomly selects its move.
- The winner is decided based on classic rules:

### 🧠 Rules:
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Draw
